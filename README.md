# Student Performance Analysis using Python

## Project Objective
To analyze student performance data and identify factors affecting academic performance using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Student performance dataset containing:
- Gender
- Race/Ethnicity
- Parental level of education
- Lunch type
- Test preparation course
- Math, Reading and Writing scores

## Key Analysis Performed
- Data Cleaning
- Feature Engineering
- Percentage Calculation
- Pass/Fail Classification
- Gender-wise Performance Analysis
- Test Preparation Impact Analysis
- Correlation Analysis
- Data Visualization

## Key Insights
- Students completing test preparation performed better
- Reading and writing scores show strong correlation
- Parental education impacts student performance

## Conclusion
This project demonstrates exploratory data analysis using Python to extract meaningful insights from educational data.
